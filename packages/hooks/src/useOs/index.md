### 参数

| 参数      | 说明         | 类型                              | 默认值                           |
|---------|------------|---------------------------------|-------------------------------|
| options | 当设置为 true 时，在组件挂载时或者浏览器环境发生变化时会重新获取操作系统信息，默认为 true。如果设置为 false，则在组件挂载时只获取一次操作系统信息 | `{ getValueInEffect: boolean }` | `{ getValueInEffect = true }` |

### 参数

| 参数 | 说明                 | 类型                                                                            | 默认值 |
| ---- | -------------------- |-------------------------------------------------------------------------------| ------ |
| OS   | 当前的操作系统信息 | `OS: 'undetermined' \| 'macos' \| 'ios' \| 'windows' \| 'android' \| 'linux'` | -      |
